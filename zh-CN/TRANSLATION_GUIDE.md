# 翻译指南 / Translation Guide

本文档为《同伦类型论》中文翻译的指导手册。

## 1. 翻译原则

### 1.1 忠实性
- 忠实传达原文的数学内容和思想
- 不随意增删原文内容
- 保持原书的章节结构

### 1.2 可读性
- 译文应通顺自然，符合中文表达习惯
- 避免生硬的直译
- 适当使用中文数学表达惯例

### 1.3 一致性
- 严格遵循术语表中的翻译
- 同一术语全书保持一致
- 定理、引理等的编号与原书一致

## 2. 术语翻译规范

### 2.1 已有标准译法的术语
直接使用学术界公认的翻译：
- type → 类型
- function → 函数
- equivalence → 等价
- homotopy → 同伦
- category → 范畴

### 2.2 HoTT 特有术语
遵循术语表 `glossary.tex` 中的翻译：
- univalence axiom → 单值公理
- higher inductive type → 高阶归纳类型
- path → 路径
- transport → 传输

### 2.3 首次出现的术语
使用 `\term{中文}{English}` 命令：
```latex
\term{同伦类型论}{homotopy type theory}是一种新的数学基础...
```

### 2.4 不翻译的内容
- 人名：保留英文原名（如 Voevodsky, Martin-Löf）
- 软件名：Coq, Agda, Lean 等
- 特定数学符号：Σ, Π, ∞ 等

## 3. LaTeX 格式规范

### 3.1 文件命名
- 章节文件与原文件同名
- 放置在 `zh-CN/chapters/` 目录下

### 3.2 编码
- 所有文件使用 UTF-8 编码

### 3.3 数学公式
- 保持原有的数学公式不变
- 行内公式使用 `$...$`
- 行间公式使用 `\[...\]` 或环境

### 3.4 交叉引用
- 保持原有的 `\label` 和 `\ref` 不变
- 确保引用正确

### 3.5 索引
- 翻译索引条目
- 保留英文原词作为辅助

## 4. 翻译工作流程

### 4.1 准备工作
1. 阅读原书对应章节，理解数学内容
2. 查阅术语表，确认专业术语翻译
3. 参考相关中文文献

### 4.2 翻译步骤
1. 从原文件复制到 `zh-CN/chapters/`
2. 逐段翻译，保持段落对应
3. 检查数学公式正确性
4. 确保交叉引用有效

### 4.3 审校
1. 自我检查：语法、术语、公式
2. 交叉审校：由其他译者审阅
3. 技术审校：确保数学内容准确

### 4.4 更新进度
- 完成后更新 `PROGRESS.md`
- 记录翻译者和审校者

## 5. 常见问题处理

### 5.1 无现成译法的术语
1. 查阅相关中文文献
2. 参考台湾、香港的翻译
3. 与其他译者讨论后确定
4. 更新术语表

### 5.2 难以翻译的表达
1. 优先保证数学含义准确
2. 可添加译注说明
3. 在译者注中解释

### 5.3 原文错误
1. 查阅原书勘误表 (errata)
2. 在翻译中更正
3. 添加译注说明

## 6. 风格指南

### 6.1 人称
- 原书的 "we" 译为 "我们"
- 保持叙述的学术风格

### 6.2 标点
- 使用中文标点符号
- 数学公式内使用英文标点
- 括号、引号使用中文格式

### 6.3 数字
- 普通数字使用阿拉伯数字
- 章节编号保持原格式

## 7. 质量标准

翻译完成需满足：
- [ ] 内容完整，无遗漏
- [ ] 术语一致，符合术语表
- [ ] 语法通顺，表达清晰
- [ ] 数学公式正确
- [ ] 交叉引用有效
- [ ] 可成功编译
